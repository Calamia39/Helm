Helm Learning Notes

Este repositorio recoge mis apuntes, pruebas y aprendizajes sobre Helm, el gestor de paquetes para Kubernetes.

¿Qué es Helm?

Helm es una herramienta que facilita la gestión de aplicaciones en Kubernetes mediante el uso de charts, que son plantillas reutilizables para desplegar recursos.

En lugar de escribir múltiples archivos YAML manualmente, Helm permite parametrizar y reutilizar configuraciones de forma más eficiente.

Objetivo del repositorio

El objetivo de este repositorio es:

Documentar lo que voy aprendiendo sobre Helm

Guardar ejemplos prácticos

Tener una referencia rápida para futuros proyectos

Experimentar con despliegues en Kubernetes
